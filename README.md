# Greencoin

Official token list for the Greencoin ecosystem.

## About Greencoin

Greencoin is a blockchain-powered renewable energy infrastructure project connected to real-world solar and energy storage assets.

The ecosystem focuses on:
- Renewable energy infrastructure
- Real-world asset (RWA) tokenization
- Green energy financing
- Decentralized energy ecosystem development
